# zzq-cli
![Npm 版本](https://img.shields.io/badge/zzq-cli_v0.0.17-blue)
## 📖 简介
用于快速创建前端工程的命令行工具
## 📚 模板
📎 快速生成 Vue3+TS+Vite+ElementPlus+Pinia+Axios+Mock或JS等 
## 🔩 安装
```bash
npm install zqc-cli -g
```
## 💡 使用

###  选择模板创建新项目
```bash
zzq-cli create 
```

###  创建指定名称并选择模板创建新项目
```bash
zzq-cli create [projectName]
```

###  更新脚手架到最新版本
```bash
zzq-cli update
```

###  查看脚手架版本
```bash
zzq-cli -v 
或
zzq-cli --version
```
